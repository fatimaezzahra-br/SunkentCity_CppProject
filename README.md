README — Sunken City


🎮 Présentation
Sunken City est un jeu 2D développé en C++ avec SFML, utilisant CMake comme système de compilation et MinGW32 comme compilateur.
Le jeu se déroule dans une ville sous-marine détruite par une catastrophe naturelle.
Le joueur incarne le dernier survivant et doit survivre en évitant des obstacles, en collectant des clés et des bonus afin de terminer tous les niveaux.

🌊 Histoire
La ville de Sunken City a été engloutie sous l’océan après une catastrophe mondiale.
Vous êtes le seul survivant.
Pour vous échapper, vous devez :
récupérer des clés 🔑
collecter des bonus ⭐
éviter les obstacles dangereux ⚠️
atteindre la sortie de chaque niveau
Si vous terminez tous les niveaux sans perdre toutes vos vies, vous gagnez le jeu.


🕹️ Gameplay
Déplacement du joueur (clavier)
Collecte de clés
Bonus (score, vie, vitesse)
Obstacles dangereux
Plusieurs niveaux
Système de victoire / défaite


⚙️ Technologies utilisées
C++
SFML (Graphics / Window / Audio)
CMake
MinGW32 (GCC)
